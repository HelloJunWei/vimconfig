# vimconfig
neovim 的設定檔案，方便自己兩台電腦使用
以及 snippets 設定

neovim安裝: https://github.com/neovim/neovim/wiki/Installing-Neovim#install-from-download

## 常用指令
1. shift + j 兩行合併成一行

## coc plugin
:CocInstall coc-flutter
:CocInstall coc-tsserver
:CocInstall coc-json
:CocInstall coc-vetur

## nvim-treesitter
:TSInstall typescript
:TSInstall query
